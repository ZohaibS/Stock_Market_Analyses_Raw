Recursive Neural Networks (RNN)

The Long Short Term Memory Algorithm

This analysis applied the Recursive Neural Network (RNN) Machine Learning algorithm (LSTM) to illustrate an application of machine learning to predict stock closing price. Of importance, RNN considers only recent data to make predictions for a very short time frame in the future.

Respective of LSTM, recent time means using the last 60-days of stock closing price data to predict the next day's closing price. Over time, if the present day can be represented by time zero, then RNN predicts stock closing on day 61.

Things to Consider

  • The model paradoxically has lower error with longer predictions. On the surface that may seem good, but is it?
  • If the predictions are only one day ahead and relatively conservative, is there any use to trying to develop trading strategies on this model?
  • Is this model an example of an instance where we can use machine learning, not to learn or derive useful insight from data, but rather to further an agenda based on visualizations containing unrealistically accurate regression lines?
